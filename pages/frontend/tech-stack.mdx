# Tech Stack

Trang này đề cập đến các công nghệ và thư viện nổi bật được sử dụng trong dự án.

Đối với các thư viên cần ít thời gian để có thể bắt đầu sử dụng, chúng sẽ không được nhắc đến.

## Core Tech Stack

- Next.js (React)
- TypeScript
- Radix UI

## Styling

- Tailwind CSS
- `clsx` - utility for className
- `tailwind-variants` - custom variants for Tailwind CSS

## Data Handling

- `SWR` - Data Fetching & Caching
- `zod` - Schema Validation
- `unified`, `retext` - Content Modification

## Testing

- Vitest - Unit & Integration Testing
- Playwright - End-to-End Testing

Cụ thể tham khảo [Tests](/frontend/tests)
