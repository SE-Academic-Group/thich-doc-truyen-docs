# Getting Started

Mong là trước khi bạn đọc phần này thì bạn đã đọc qua các phần tổng quan trước đó. Nếu chưa, bạn có thể xem lại ở [Trang chủ](/), hoặc [Kiến trúc](/architecture).

## Yêu cầu

Đây là yêu cầu về các công nghệ cho việc phát triển phần backend của dự án **Thích Đọc Truyện**.

- [Java](https://www.java.com/en/download/) JDK 17+
- [Maven](https://maven.apache.org/download.cgi) 3.8.x +

## Cài đặt

Chi tiết xem tại [GitHub Repo](https://github.com/SE-Academic-Group/thich-doc-truyen-backend)

Để cài đặt môi trường phát triển, bạn có thể thực hiện các bước sau:

1. **Clone repository về máy**

```bash copy
git clone https://github.com/SE-Academic-Group/thich-doc-truyen-backend.git
```

2. **Build dự án thành file .jar**

Sử dụng maven trong máy của bạn.
```bash copy
mvn clean package
```
Sử dụng maven wrapper trong dự án có sẵn.
```bash copy
./mvnw clean package
```
3. **Chạy ứng dụng**

Sử dụng maven trong máy của bạn.
```bash copy
mvn spring-boot:run
```
Sử dụng file .jar đã build trước đó.
```bash copy
java -jar target/novel-aggregator-0.0.1-SNAPSHOT.jar
```

4. **Mở trình duyệt và truy cập vào http://localhost:8080/swagger-ui/index-html**. Đây là 1 trang tài liệu được hỗ trợ tạo ra từ **Swagger OpenApi**. Bạn có thể dùng nó để hiểu rõ cách thức hoạt động và nhận dữ liệu từ api.
