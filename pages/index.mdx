# Thích Đọc Truyện

**Thích Đọc Truyện** là một trang web đọc truyện online, dữ liệu truyện được cào từ các nguồn truyện online như [Truyện Full](https://truyenfull.vn/), [Tang Thư viện](https://truyen.tangthuvien.vn/) (nguồn hơi dỏm), và [Thích Truyện](https://thichtruyen.vn/).

Đây là đồ án cuối kỳ môn học Thiết kề phần mềm tại Khoa CNTT trường Đại học Khoa học Tự nhiên (fit@hcmus).

## Mục đích của đồ án môn học

- Áp dụng các kiến thức đã học trong môn học Thiết kế phần mềm.
- Sử dụng các kỹ thuật thiết kế phần mềm để phát triển một phần mềm dễ mở rộng (scalable).
- Sử dụng các coding principles như SOLID, DRY, KISS, YAGNI, ... để viết code dễ đọc, dễ bảo trì.
- Áp dụng các coding conventions như naming, indentation, ... để code đồng nhất trong cùng một nhóm.
- Sử dụng một số design patterns phổ biển.
- Áp dụng các kỹ thuật testing để đảm bảo chất lượng phần mềm.
- Quản lý dự án bằng Git và GitHub (và GitHub Actions).
- Áp dụng một số kỹ thuật refactoring để cải thiện codebase theo thời gian.
